System Requirements

  • Rhino V7 or later
  • Internet connection
  • Windows 10+ / MacOS X 14+, 32GB RAM

Installing sphereneRHINO V2

How to install (2 min)​

  1. In Rhino, run PackageManager
  2. Search sphereneRHINO → choose the 2.x.x package → Install ▸ Apply.
  3. Restart Rhino.

Toolbar Activation​

Rhino 8​

  1. Open Rhino3D and enter Toolbar in the command line.
  2. Select spherene from the dropdown menu
  3. Check the spherene V2 checkbox

Close the menu and position the toolbar to your liking.

Rhino 7​

warning

We are working on providing you the Rhino 7 toolbar for spherene V2 soon. Until then, please use the commands (Just start writing "sphe" to see all available commands) or use the V1 toolbar (Some tools like the sphe_inspector are not in the toolbar but can be added manually) sphereneRHINO V1 Toolbar for Rhino 7 - Right click save

  1. Download the Toolbar (Right click, Save): sphereneRHINO V2 7 Toolbar (If the link does not work, please try again in a few days, it might not be yet uploaded)
  2. Drag & drop the downloaded .RUI file from your file explorer into Rhino 7

Position the toolbar to your liking.

Updating spherene​

For updates, revisit the packagemanager, select the Installed Tab, select sphereneRHINO and check for new versions, install if available. Restart Rhino after every change.

If you like you can read the release notes using the command sphe_about or the menu Spherene → Utilities → Release Notes. Here is a direct link to the release notes
